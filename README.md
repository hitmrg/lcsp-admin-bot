# lcsp-admin-bot
bot discord, gestion administrative lcsp (laboratoire cybersécurité supinfo paris)

### Commandes disponibles:

**Admin:**
- `/setup` - Initialiser le serveur
- `/purge [nombre]` - Nettoyer les messages
- `/announce [titre] [message]` - Faire une annonce

**Membres:**
- `/membre_add` - Ajouter un membre
- `/membre_info` - Voir les infos
- `/membre_update` - Modifier un membre
- `/membre_delete` - Supprimer un membre
- `/membres` - Liste des membres

**Réunions:**
- `/meeting_create` - Créer une réunion
- `/appel [id]` - Faire l'appel
- `/meetings` - Prochaines réunions

**Rapports:**
- `/stats` - Statistiques générales
- `/rapport` - Rapport d'activité
- `/export` - Exporter en CSV
